# エージェント間共有メモ (RALPH lessons)

このファイルは `.harness/master.py` がエージェントを起動する際に、システムプロンプトに合わせて読み込まれる「直近の教訓」プールである。
`AGENTS.md` の `## 失敗から学んだこと (自動生成)` が SARIF ベースの自動学習なのに対し、`lessons.md` は人間または master.py が手動で書く小さなメモ。

## 書き方

- 1 件 = 1 行 = 1 教訓
- 短く、具体的に。例: `- F# module 初期化 startup-code 型は ArchUnit テストで除外する必要あり`
- 古くなった教訓は削除して構わない (履歴は git log に残る)
