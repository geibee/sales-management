# 記入済みサンプル: 値引き承認機能（架空）

**これは書き方を示すための架空の依頼**であり、実際の開発対象ではない。実在の依頼は `specs/requests/` に置く。

このサンプルは「誰が・どこまで書くか」の全体像を示す。記入者は 3 種類に分かれる:

| 記入者 | 書く場所 | このサンプルでの該当箇所 |
|---|---|---|
| **顧客 + 起票者**（合意形成の対象） | 業務の言葉の表と例 | `00` 全体、`10` §1・§3〜§10、`50` §1〜§5 |
| **エンジニア**（顧客レビュー不要、人間の責任範囲） | 型スケッチ、契約差分とその承認 | `10` §2、`20` §6 の openapi 差分と承認欄 |
| **AI**（③ゲート先行フェーズ） | 各ファイル末尾の品質ゲート化対応表のテスト ID | 各「品質ゲート化対応表」の記入済みセル |
